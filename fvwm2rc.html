# RandomPlacement

# functions to move to next (window, page, desk)
AddToFunc  next-dsk-raise   I Desk  -1  0  0  4
AddToFunc  prev-dsk-raise   I Desk   1  0  0  4

AddToFunc  next-pg-raise    I Scroll  +100000 +0
AddToFunc  prev-pg-raise    I Scroll  -100000 +0

AddToFunc  next-win-raise   I Next [CurrentPage  !iconic] Focus
+                           I Current Raise
+                           I Current WarpToWindow 20 20

AddToFunc  prev-win-raise   I Prev [CurrentPage  !iconic] Focus
+                           I Current Raise
+                           I Current WarpToWindow 20 20

+                               I Current Iconify -1
+                               I Current Raise
+                               I Current WarpToWindow 5p 5p

+                               I Current Iconify -1

+                               I Current WarpToWindow 5p 5p

# 1. <alt><tab>                 next window on current page
# 2. <alt><shift><tab>          previous window on current page
# 3. <ctrl><alt><tab>           next window (will de-iconify) on cur page
# 4. <ctrl><alt><shift><tab>    previous window (will de-iconify) on cur pg
# 9. <alt><click-3> or <F12>    Raise-lower the window at current 
#                               cursor-position, like Sun's "Front" key.
